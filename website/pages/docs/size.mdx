# size

The `size` middleware provides data to change the size of the
floating element so it doesn't overflow the available space.

<div className="flex flex-col gap-4">
  <Chrome center scrollable className="h-96">
    <Floating
      tooltipStyle={{
        height: 200,
        overflow: 'hidden',
        maxHeight: 0,
      }}
      middleware={[{name: 'size', options: {padding: 5}}]}
    >
      <div className="grid place-items-center text-gray-50 w-32 h-32 bg-blue-600">
        Scroll
      </div>
    </Floating>
  </Chrome>
</div>

This can be used in conjunction with `flip` to make the floating
element prefer its initial `placement` as much as possible, until
it reaches a minimum acceptable size.

<Chrome center scrollable className="h-96">
  <Floating
    tooltipStyle={{height: 200}}
    middleware={[
      {
        name: 'flip',
        options: {
          rootBoundary: 'document',
          fallbackStrategy: 'bestFit',
        },
      },
      {
        name: 'size',
        options: {rootBoundary: 'document', padding: 5},
      },
    ]}
  >
    <div className="grid place-items-center text-gray-50 w-32 h-32 bg-blue-600">
      Scroll
    </div>
  </Floating>
</Chrome>

If your floating element's content cannot be resized such as in
the example, you can make the floating element scrollable with
`overflow: scroll` (or `auto`).

## Usage

```js
import {computePosition, size} from '@floating-ui/dom';

computePosition(referenceEl, floatingEl, {
  middleware: [size()],
}).then(({middlewareData}) => {
  const {width, height} = middlewareData.size;

  // If your element has a border, ensure your CSS is using
  // `box-sizing: border-box`
  Object.assign(floatingEl.style, {
    maxWidth: `${width}px`,
    maxHeight: `${height}px`,
  });
});
```

## Data

These dimensions represent how large the floating element can be
before it will overflow.

```ts
type Data = {
  width: number;
  height: number;
};
```

## Options

```ts
type Options = DetectOverflowOptions;
```

### ...detectOverflowOptions

All of [detectOverflow](/docs/detectOverflow#options)'s options
can be passed. For instance:

```js
size({padding: 5}); // 0 by default
```
